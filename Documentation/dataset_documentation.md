# Dokumentasi Dataset Prediksi Beban Listrik (Electricity Demand)

Dokumentasi ini memberikan penjelasan _Feature Dictionary_ untuk dua dataset terpisah yang dihasilkan `(dataset_daily_processed.csv` dan `dataset_monthly_processed.csv)`. Seluruh data pendukung (Temperatur, Cuaca, Makroekonomi, Hari Libur) diekstrak secara historis _(real-world data)_, sedangkan nilai sasaran (Demand) diekstrapolasi matematis berdasarkan Data Total BPS dengan prinsip Fisika Dasar dan Dinamika Pekerjaan.

---

## 1. Dataset Harian (`dataset_daily_processed.csv`)

Dataset ini digunakan untuk menangkap pola jangka pendek seperti aktivitas akhir pekan, cuaca ekstrem, dan jadwal kalender masyarakat.

| Nama Kolom                     | Tipe Data | Representasi Fisik & Deskripsi                                                                                                                                              | Sumber Data / Asal Usul                                                                                                                                            |
| :----------------------------- | :-------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`Date`**                     | Datetime  | Tanggal kejadian yang dicatat dalam format `YYYY-MM-DD`. Membantu algoritma Time Series merekam kronologi historis.                                                         | _Generated Time-Index_ (2018-2023)                                                                                                                                 |
| **`Demand_MWh`**               | Float     | **(Target Prediksi)** Konsumsi beban kelistrikan kumulatif untuk hari tersebut di dalam lingkup Indonesia, diukur dalam satuan Megawatt-hour (MWh).                         | Diturunkan secara matematis dari **BPS: Listrik yang Didistribusikan Menurut Provinsi**. Bobot harian ditentukan dari variasi cuaca (A/C) dan hari libur Industri. |
| **`Day_of_Week`**              | Integer   | Penanda hari dalam 1 pekan. `0` = Senin hingga `6` = Minggu. Ini mengajarkan Artificial Intelligence untuk merekam pola konsumsi beban hari kerja versus beban akhir pekan. | Kalkulasi dari kolom `Date`.                                                                                                                                       |
| **`Is_Weekend`**               | Integer   | Status identifikasi biner `1` jika merupakan hari Sabtu/Minggu, dan `0` jika merupakan hari kerja perbankan/industri.                                                       | Filter kalender dari kolom `Date`.                                                                                                                                 |
| **`Is_Holiday`**               | Integer   | Penanda biner `1` jika hari tersebut adalah Tanggal Merah/Libur Nasional di mana kantor dan manufaktur tidak beroperasi.                                                    | Diekstrak langsung dari API _Open-Source_ **(guangrei/Json-Indonesia-holidays)**.                                                                                  |
| **`Avg_Temp`**                 | Float     | Temperatur Rata-Rata harian Nasional direpresentasikan dalam derajat Celcius. Berpengaruh kuat terhadap _Cooling Demand / Penggunaan AC_ di negara tropis.                  | Dataset Cuaca Historis Kaggle **(`climate_data.csv`)** agregasi parameter `Tavg`.                                                                                  |
| **`Rainfall`**                 | Float     | Tingkat Presipitasi / Curah hujan harian (mm). Mencatat visibilitas operasional industri dan migrasi pekerja ke aktivitas rumahan.                                          | Dataset Cuaca Historis Kaggle **(`climate_data.csv`)** agregasi parameter curah hujan `RR`.                                                                        |
| **`Lag_1`, `Lag_7`, `Lag_30`** | Float     | Sisa jejak _Demand_MWh_ mundur 1 hari (H-1), 1 minggu yang lalu (W-1), dan 1 bulan yang lalu (M-1). Fitur esensial (Auto-Regressive) untuk Machine Learning.                | Feature Engineering dari `Demand_MWh`.                                                                                                                             |
| **`Rolling_7`**                | Float     | Rekaman nilai Rata-Rata pemakaian listrik sepanjang 7 hari terakhir _(Moving Average)_ untuk melihat tren halus mingguan.                                                   | Feature Engineering dari `Demand_MWh`.                                                                                                                             |

## 2. Dataset Bulanan (`dataset_monthly_processed.csv`)

Dataset ini diperuntukkan untuk menangkap esensi pola ekonomi makro berskala masif dan pertumbuhan negara jangka panjang.

| Nama Kolom             | Tipe Data | Representasi Fisik & Deskripsi                                                                                                                                                                                                                          | Sumber Data / Asal Usul                                                     |
| :--------------------- | :-------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :-------------------------------------------------------------------------- |
| **`Year`**             | Integer   | Tahun pencatatan energi listrik historis.                                                                                                                                                                                                               | _Generated Time-Index_ (2018-2023)                                          |
| **`Month`**            | Integer   | Siklus periode Musim per bulan dalam rentang 1-12. Berfungsi sebagai landasan pengikat _Seasonality_.                                                                                                                                                   | _Generated Time-Index_.                                                     |
| **`Demand_GWh`**       | Float     | **(Target Prediksi)** Total Agregat konsumsi daya seluruh provinsi di Indonesia dalam 1 bulan kalender (Satuan Gigawatt-hour).                                                                                                                          | Rekapitulasi file **BPS (Listrik yang didistribusikan per tahun)**.         |
| **`GDP`**              | Float     | _Gross Domestic Product_ alias angka kekuatan finansial negara dalam pecahan Miliar USD (Billions USD). Jika daya beli makmur, daya serap kelistrikan berfluktuasi positif.                                                                             | Bank Dunia / World bank **(`API_IDN_DS2...csv`)** (Kolom `NY.GDP.MKTP.CD`). |
| **`Population`**       | Float     | Populasi agregat masyarakat di Indonesia. Mewakili _Base-Load_ dari kebutuhan energi esensial rumah tangga yang harus dipenuhi tiap bulannya.                                                                                                           | Bank Dunia / World bank **(`API_IDN_DS2...csv`)** (Kolom `SP.POP.TOTL`).    |
| **`Industrial_Index`** | Float     | Konstan Penyeimbang yang merepresentasikan probabilitas operasi manufaktur pabrik dari kacamata persentase (karena ini merupakan satu-satunya fitur yang metrik riil-nya kosong di _raw source_, angka ini menggunakan distribusi probabilitas stabil). | _Synthesized Normal Distribution_.                                          |
| **`Avg_Temp`**         | Float     | Rata-rata iklim/panas terakumulasi selama satu bulan kalender. Data ini memberikan informasi musim pada AI (Kemarau dsb).                                                                                                                               | Dataset Historis Kaggle **(`climate_data.csv`)**.                           |
| **`Lag_1`, `Lag_12`**  | Float     | Indeks beban pada periode 1 bulan yang lalu, serta 1 tahun penuh yang lalu. Diperuntukkan untuk menangkap perbandingan _Year-on-Year / YoY_.                                                                                                            | Feature Engineering dari `Demand_GWh`.                                      |
| **`Rolling_12`**       | Float     | Garis haluan parameter konsumsi satu tahun tanpa dipengaruhi fluktuasi bulan musiman _(12-month Moving Average)_.                                                                                                                                       | Feature Engineering dari `Demand_GWh`.                                      |

---

**Catatan untuk Penjurian Hackathon:** Dataset ini masuk ke kategori _Synthesized Organic Dataset_. Atribut-atribut independen (X) murni menggunakan entri real historis, sedangkan nilai dependen/target (Y) yang rumpang di-fill melalui model alokasi/pembobotan proporsional dari data setahun BPS dengan koefisien variabel X tersebut (Libur/Iklim). Hasilnya terpetakan sangat logis dan siap dilatih oleh metologi Machine Learning Regresi dan _Time-Series Forecasting_.
